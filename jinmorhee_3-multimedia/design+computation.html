Design Computation is work about the geometry, algorithm, and data structure with computing, especially Python coding. In this project, individual shape of geometry is not focused. But the it focused more on representation of the algorithm through geometry. Shape grammars, stack, KD-Tree, circle-packing, fractals, trees, search algorithms, boids, L-system, graphs, optimization, discrete mathematical models and agent-based models, etc. visualization of the algorithma and data structure via geometry can help designers not only understand the programming but also increase their intuition about the programming and computational design.
